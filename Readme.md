# Usage

`./dorfctl.py [enable|disable|toggle|do] <device>`

Use 'do' for shutdown and wakeup


# Examples
Enable something

`./dorfctl.py enable regal1`

`./dorfctl.py enable tische`

Shutdown

`./dorfctl.py do shutdown`

`./dorfctl.py do wakeup`

Use presets

`./dorfctl.py Lounge hell`
